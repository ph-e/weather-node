# Node Weather
## Application Description
With this application you can find out the current weather in the city of your choice.
## Installation requirements
- [Node js](https://nodejs.org/en/download)<br>
## Installation process
- Clone this repo
```
git clone https://github.com/ph-e/weather-node
```
- Open the project folder and write the command
```
npm i
```
## Launching the application
- Write the following command to get acquainted with the functionality
```
npm start -- -h
```

- Don't forget to specify the token for the application robot. You can get it on the website:
```
https://openweathermap.org/
```
